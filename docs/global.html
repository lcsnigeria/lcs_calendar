(constant) existingCSS

Adds an external CSS stylesheet for the `lcs_calendar` library to the document's section, if not already present. This function first selects all existing elements with an `href` attribute containing "lcs_calendar". If none of such element is found, it creates a new element pointing to the `lcs_calendar` stylesheet and appends it to the document's section. The css file is located at src/ (the main and unminified version) and dist/ (the minified version)

defaultConclusionCallback()

Default callback function for handling the conclusion of calendar interactions in "input" mode. This function validates year, month, and date selections, then formats and assigns the selected date to input fields. If all required selections are made, the calendar instance is removed from the DOM after completion.
